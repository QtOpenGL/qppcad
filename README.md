# qpp::cad
### current status - under construction
 molecule and crystal editor written in c++
### How to build
  Init submodule "qpp" first.

### Dependecies
1. qpp::core
2. glfw3
3. imgui
4. freetype
5. nativfiledialog

### Current features

1. Manipulate the atoms list(molecules or periodic crystals)
